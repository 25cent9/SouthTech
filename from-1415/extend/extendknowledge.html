Internet Course- MGMT 102

This week in class, we learned many of the key terms associated with Internet technology. We would like to provide you with a list of terms and definition that you can use as a study guide.

Key Elements

Internet
The Internet is a worldwide collection of computer networks linked together. The Internet is a public access network and is not password protected. Satellites, fiber optics, and phone lines are some devices used for connection.
World Wide Web(WWW)
This is the part of the Internet that consists of a collection of linked documents. Web page developers create pages of text together with HTML code that will display in a browser.
HTTP
The term stands for Hypertext Transfer Protocol. It is a set of rules that are used for exchanging text, graphic, sound, and other multimedia files. The linked documents, or pages of information, on the Web are known as Web pages.
